La interfaz esta compuesta or un formulario que contiene la información necesaria para registrar una venta:

1.- Pagado por (cliente).-En este campo se debe introducir el nombre del cliente quien realizara la compra, este cliente debera estar registrado en el sistema, si no esta registrado el sistema redirigirá al usuario al modulo de registro para registrarlo(ver registro de usuarios pag***), no se podrá registrar la venta hasta después de haberlo registrado. El sistema reconocerá el nombre personal del cliente, y al seleccionarlo apareccera un recuadro en el cual podrá verificar sí los datos del cliente son correctos antes de registrar la venta (recuadro rojo).

Total de la venta.- En este campo se debera digitar la cantidad total de la venta. (lo que el cliente pagará).

Importe de la Venta (letras).- En este campo se debera escribir con letras el importe que se ha dijitado en la cantidad total de la venta.

Fecha entrada.- en este campo se debe seleccionar la fecha en la se se aplicara el servicio.

Fecha regreso.- en este campo se debera seleccionar la fecha en la que termina el servicio.

Descriopcion.- En este campo se debera dar una bereve descripccion con los datos importantes del servicio.

Boton de registrar venta.- Al pulsar el boton la evnta se registrar y se encontrara disponible en la seccion de abono
