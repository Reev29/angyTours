Ingresar Ventas


Las ventas que el sistema permite ingresar son Ventas Individuales y Ventas Grupales, para ambos casos se desplegará un formulario en el que se deberá ingresar los datos correspondientes. Hay que tomar en cuenta que, para realizar una venta en cualquiera de las dos modalidades, deberá estar registrado el cliente (Ver sección Registro Cliente). Los datos mostrados en el formulario son los siguientes:





Los campos de “Pagado por el Cliente”, “Nombre del Titular del Grupo”, “Hotel” y “Operadora Mayorista” se autocompletarán cuando el usuario digite en él, teniendo en cuenta que, para que esto funcione, se deberá tener un registro previo de los campos antes mencionados

Para los campos Pagado por el Cliente y Nombre del Titular del Grupo serán los clientes que pertenezcan a la base de datos del sistema en ese momento (Ver sección Registro Cliente), por otra parte, el campo Hotel tomará los datos de los Hoteles registrados en ese momento (Ver sección Hoteles). El campo Operadora Mayorista ya estará precargado con las operadoras que cuenta la Agencia.





Abonar Ventas



Los abonos solo se podrán efectuar en las ventas que ya se encuentren registradas en el sistema, tanto para Ventas Individuales como para Ventas Grupales.

  • 1. Se deberá introducir el nombre del titular de la venta en la modalidad correspondiente (Ventas Individuales – Abonos Individuales, Ventas Grupales – Abonos Grupales)
  • 2. Elegir en la tabla la venta a la que se desea realizar el abono
  • 3. Digitar la cantidad deseada a abonar.
  • 4. Se mostrará un mensaje donde indique que el abono se efectuó correctamente
  • 5. Imprimir el comprobante, si así se desea o regresar a la interfaz principal.





Borrar Ventas



Las ventas registradas podrán borrarse del sistema tanto para Ventas Individuales como para Ventas Grupales.

  • 1. Se deberá introducir el nombre del titular de la venta en la modalidad correspondiente (Ventas Individuales – Borrar Ventas Individuales, Ventas Grupales – Borrar Ventas Grupales)
  • 2. Elegir en la tabla la venta a la que se desea borrar.
  • 3. Decidir si la venta realmente se desea borrar o regresar a la interfaz principal.
  • 4. Si se seleccionó la opción de borrar, el sistema alerta que la venta ya se ha borrado, y se regresa automáticamente a la interfaz principal.





Generar Cupones Ventas



Para poder generar cupones desde el sistema, se deberá tener en cuenta que la venta ya debió de estar concluida, en caso contrario, el sistema no mostrará la venta para generar el cupón.

  • 1. Se deberá introducir el nombre del titular de la venta en la modalidad correspondiente (Ventas Individuales – Cupón Individual, Ventas Grupales – Cupón Grupal).
  • 2. Elegir en la tabla la venta a la que se desea generar el cupón.


Para ventas individuales:

  • 3. Llenar el formulario con los campos requeridos
  • 4. Dar clic en el botón Generar Cupón
  • 5. El sistema alertará que el cupón se ha generado con éxito
  • 6. Se podrá descargar el cupón o regresar a la interfaz principal


Para ventas grupales:

  • 3. Se deberá introducir los pasajeros del grupo uno a la vez.
  • 4. Ya ingresados todos los pasajeros correspondientes al grupo, se podrá imprimir el cupón de todos, teniendo en cuenta que será uno a la vez.
  • 5. Se podrá borrar los pasajeros del sistema que no pertenezcan al grupo.
  • 6. Se podrá regresar a la interfaz principal.





Pagos



En esta sección podrá agregar/subir fichas de depósito al sistema y, también, podrá ver las fichas almacenadas por los clientes u otros usuarios.

Nuevo Voucher:



Se desplegará un formulario con la información necesaria para identificar el pago

Dicho pago deberá ser verificado, para después abonarse a la cuenta a la que esté asignado (Ver sección Ver Voucher).



Ver Voucher:



Se desplegará una tabla que muestra los pagos que se han realizado con fichas de depósito. Podrá dar clic al botón “Ver Voucher” para visualizar el documento en cuestión.





Pagos Pendientes



Se podrán elaborar comprobantes de pagos para las ventas cerradas. Hay que tomar en cuenta, que, para que una venta pueda ser pagada, deberá haberse liquidado por el cliente.

  • 1. Deberá elegir el tipo de venta “Ventas” o “Grupos”.
  • 2. Elegir en la tabla la venta a la que se desea generar el comprobante de pago.
  • 3. Llenar el formulario correspondiente.
  • 4. Dar clic en el botón registrar pago.
  • 5. Podrá descargar el Voucher de pago o, de lo contrario, regresar a la interfaz inicial.





Pagos Cerrados



Se provee una tabla informativa de los pagos que se han realizado mostrando desde los más recientes hasta los más antiguos.





Nueva Promoción



En esta sección podrá dar de alta a nuevas promociones:

Se desplegará un formulario con los datos necesarios para registrar la promoción.

Campo Botón Paypal: En este campo se deberá introducir el código que se habrá generado en página de PayPal.





Administrar Promoción



Las promociones existentes aparecerán en una tabla que muestra dos acciones diferentes: “Actualizar” y “Borrar”.

Actualizar: Se mostrará la información actual así como también campos para editar dicha información.

Borrar: Al presionar el botón se borrará la venta por completo del sistema y se mostrará una alerta notificando dicha acción.





Nuevo Hotel



Este apartado, se encarga de registrar lo hoteles en la base de datos del sistema.

Se desplegará un formulario para introducir los datos e imágenes del hotel.





Administrar Hotel



Se desplegará una tabla en la cual aparecerán los hoteles existentes en el sistema, se dispondrán de tres opciones por cada hotel “Detalles”, “Actualizar”,“Borrar”.

  • Detalles: Muestra la información del hotel que se encuentra en la base de datos.

  • Actualizar: Deberá introducir la información nueva en el campo editable.

  • Borrar: Se eliminará toda la información del hotel del sistema.





Agregar viajero



Se deberá complementar la información requerida en el formulario, para los campos:

Nombre de usuario, Teléfono y Correo: La información en este campo no podrá ser utilizada por dos usuarios la vez.

En caso de que ya se encuentren registrados alguno de estos campos, el sistema alertará y no realizará el registro.





Agregar Agente de ventas



Se desplegará un formulario con la información necesaria para el registro de un “Agente de Ventas”. Al igual que en el registro de viajero (Ver sección Agregar Viajero), hay campos que no deberán duplicarse. El sistema informará sobre estas situaciones.





Bitácoras



Esta sección contiene 2 sub apartados: “Ventas Individuales”, “Ventas Grupales”.

Se desplegará una tabla con los movimientos realizados tanto para Ventas Individuales, como para Ventas Grupales.

Este apartado permite calcular el total de ventas realizadas en un intervalo de tiempo, para realizar esta acción se deberá:





  • 1. Introducir un intervalo de tiempo en el que constarán las ventas que se deseen calcular.
  • 2. Dar clic en el botón Calcular.
  • 3. Se desplegará la cantidad exacta de los ingresos o egresos, según se requiera del intervalo de tiempo antes proporcionado.
  • 4. Se podrá descargar un reporte dando clic en el botón Imprimir Reporte.
